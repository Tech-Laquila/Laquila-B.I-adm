## Technical Context

### Fix 1: Lazy load do `DashLineChart`

Nas pĂˇginas que usam o componente, substituir o import direto por `next/dynamic`:

```ts

// DEPOIS

    () => import("@/components/dashboard/line-chart").then(m => ({ default: m.DashLineChart })),
    {
        loading: () => (
            <div className="bg-[#0d0d0d] border border-[#1f1f1f] rounded-xl h-72 animate-pulse" />

        ssr: false,
    }
);
```

O `ssr: false` Ă© correto aqui pois o grĂˇfico usa `ResponsiveContainer` que depende de dimensĂµes do DOM.

### Fix 2: Promise.all() em `vendedores.ts`
